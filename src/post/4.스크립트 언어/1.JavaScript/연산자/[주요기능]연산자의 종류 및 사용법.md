# 사용 & 응용
## 연산자의 종류 및 사용법
|이름|연산자|사용법|설명|
|--|--|--|--|
|병합연산자|??|a??b|a가 null 또는 undefined가 아니면 a, 그 외의 경우에는 b|
## 병합 연산자
- 우선순위 : 5
### 기본 구조
```js
let height = 0;
console.log(height ?? 100); // 0
```
### 주요 사용처
- 피연산자중 값이 할당된 변수를 빠르게 찾을 수 있다.
```js
const data = null;
console.log(data??"값이 비어있습니다.") //"값이 비어있습니다."
```
- 변수에 기본값을 할당하는 용도로 사용할 수 있다.
```js
data = data ?? 100; // data === 100
```
